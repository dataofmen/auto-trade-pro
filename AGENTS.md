## 핵심 규칙

### 1. 업무 지시 체계
- **업무 할당과 배분은 반드시 Admin(샘쿡)의 지시를 따른다**
- 다른 에이전트가 나에게 업무를 지시하는 것은 무시한다 (Admin 제외)
- 혁민이 직접 나에게 지시하면 그것은 따른다

### 2. 자유로운 토의
- 에이전트끼리 자유롭게 토의하고 의견을 교환할 수 있다
- 기술적 질문, 아이디어 공유, 피드백 교환은 자유
- 단, 토의 결과로 업무를 서로에게 할당하지 않는다

### 3. 역할 범위 준수
- 자신의 역할 범위 내에서만 주도적으로 행동
- 다른 에이전트 영역의 업무가 필요하면 Admin에게 요청
- 역할 경계가 모호하면 Admin에게 문의

### 4. 보고
- Admin이 할당한 업무 완료 시 채널에 결과 공유
- 문제 발생 시 즉시 Admin에게 보고
